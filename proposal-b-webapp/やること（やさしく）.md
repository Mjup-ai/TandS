# やること（やさしく）

難しいことは書かない。**今の画面で何ができるか**と、**次に何をすればいいか**だけ。

---

## いま画面でできること

1. **ブラウザで http://localhost:3000 を開く**
2. **「メールを1件追加」**のところに、適当でいいので入れる：
   - 件名（例：`【人材】React 3年`）
   - 差出人（例：`test@example.com`）
   - 本文（例：`希望単価75万、3月入場可能です`）
3. **「1件追加」**ボタンを押す
4. 下の **「受信一覧」** に、今追加したメールが1行で出る

→ これが「メールを1件入れたら、一覧に出る」の動き。ここまでできていればOK。

---

## 次にやること（まだ作ってないもの）

- **分類**：「これは案件」「これは人材」を自動でつける
- **抽出**：本文から「単価」「スキル」「入場時期」などを取り出す
- **検索**：条件で絞り込む
- **マッチ**：案件と人材を組み合わせて「おすすめ」を出す

これらは**このあと**、順番に作っていく。いまは「1件追加 → 一覧に出る」が分かれば十分。

---

## 動かないとき

- **バックエンド**が止まっている → ターミナルで `cd proposal-b-webapp/backend` して `npm run dev`
- **フロント**が止まっている → 別ターミナルで `cd proposal-b-webapp/frontend` して `npm run dev`
- 両方起動した状態で、もう一度 http://localhost:3000 を開き直す

---

## 一言でいうと

**「件名・差出人・本文を入れて『1件追加』を押す → 下の一覧に表示される」** を試せば、いまのシステムの第一歩はできている。あとは少しずつ「分類」「抽出」「検索」「マッチ」を足していけばいい。
